# Literary Recommendation Agent

A RAG-based agent that recommends books based on your reading history and checks real-time prices across Brazilian bookstores.

## How it works

1. You input books you've already read
2. The agent fetches their metadata via the Open Library API and builds a similarity query
3. A ChromaDB vector store retrieves candidate books from a pre-built catalog
4. Candidates are enriched with full descriptions via the Open Library Works API
5. Prices are checked live on Mercado Livre, Amazon BR, and Estante Virtual
6. A local Ollama LLM ranks the candidates and writes a justification for each recommendation in Portuguese

## Requirements

- Python 3.11+
- [Ollama](https://ollama.com) running locally

## Setup

```bash
# 1. Clone and enter the project
cd felipe-amorim-de-araujo/projeto-1

# 2. Create and activate a virtual environment
python -m venv .venv
source .venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Pull the default LLM
ollama pull qwen2.5:7b

# 5. Configure environment (optional — to change the model)
cp .env.example .env

# 6. Build the book catalog (run once)
python src/catalog_builder.py

# For a quick test build (~20 books per subject):
python src/catalog_builder.py --limit 20
```

## Running

```bash
streamlit run src/app.py
```

Open [http://localhost:8501](http://localhost:8501).

## Project structure

```

├── src/
│   ├── app.py              # Streamlit UI
│   ├── agent.py            # Orchestration: RAG → enrichment → prices → LLM
│   ├── book_fetcher.py     # Open Library Search + Works API
│   ├── catalog_builder.py  # Populates ChromaDB from Open Library Subjects API
│   ├── rag.py              # ChromaDB vector store wrapper
│   ├── price_checker.py    # Mercado Livre API + Amazon BR + Estante Virtual
│   └── wishlist.py         # Local wishlist (JSON)
├── data/
│   ├── chroma_db/          # Vector store (generated by catalog_builder)
│   └── wishlist.json       # Wishlist (generated at runtime)
├── .env.example
└── requirements.txt
```

## Configuration

| Variable | Default | Description |
|---|---|---|
| `OLLAMA_MODEL` | `qwen2.5:7b` | Ollama model to use for ranking |

## Price sources

| Store | Method |
|---|---|
| Mercado Livre | Official API (`api.mercadolibre.com`) |
| Amazon BR | Scraping (best-effort — may be blocked) |
| Estante Virtual | Scraping |

## Notes

- The catalog build is a one-time step. Re-run it if you want to refresh the book catalog.
- Mercado Livre prices are the most reliable since they come from an official API.
- Amazon BR scraping may intermittently fail due to anti-bot measures.
