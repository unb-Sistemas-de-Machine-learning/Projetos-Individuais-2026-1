# Evidencias

Esta pasta deve armazenar evidencias de execucao do workflow, por exemplo:

- capturas de tela do formulario e das rotas no n8n;
- logs de execucao por caso de teste (alta, media e baixa urgencia);
- comprovacao de envio de email;
- registros de persistencia no Data Table.

## Nomenclatura sugerida

- `evidence-t1-alta-urgencia.png`
- `evidence-t2-media-urgencia.png`
- `evidence-t3-baixa-confianca.png`
- `evidence-persistencia-datatable.png`

## Checklist rapido

- [ ] Pelo menos 1 evidencia para cada rota de urgencia
- [ ] Pelo menos 1 evidencia de persistencia
- [ ] Pelo menos 1 evidencia de acao de saida (email)
