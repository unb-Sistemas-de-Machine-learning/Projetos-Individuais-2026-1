# Projeto 3: Multi-Agent Red Team Framework

Este projeto apresenta um framework automatizado de cibersegurança (Red Team) orquestrado pelo **n8n**, utilizando inteligência artificial para planejar, validar e reportar vulnerabilidades.

## A Solução Final (Maturidade C)
A implementação consolidada deste projeto é a **Solution C**, um sistema multiagente autônomo capaz de orquestrar o ciclo completo de Red Team de forma iterativa e resiliente.

## Estrutura do Repositório
- `docs/`: Documentação completa de engenharia (Mission Brief, Mentorship Pack, ADRs).
- `solutions/`: Implementação das três camadas de maturidade (A, B e C), sendo C a entrega final.
- `tests/`: Payloads para validação técnica dos fluxos.
- `relatorio-entrega.md`: Relatório técnico final consolidado.

## Como Executar
1. Importe o JSON de fluxo da **Solution C** para o seu n8n.
2. Configure as credenciais das APIs (VirusTotal, Gemini).
3. Utilize os payloads de teste para disparar os fluxos.
4. Consulte os logs de execução para auditoria.
