# Solution B — RAG simples com base de conhecimento

Abordagem que usa uma base local de politicas para recuperar o contexto mais aderente e orientar a classificacao.

## Caracteristicas

- maior contextualizacao que a Solution A
- melhora consistencia de resposta
- requer manutencao da base de conhecimento

## Execucao

```bash
python solutions/solution-b/main.py "Preciso da segunda via do boleto da mensalidade. Meu RA e 211061529"
```
