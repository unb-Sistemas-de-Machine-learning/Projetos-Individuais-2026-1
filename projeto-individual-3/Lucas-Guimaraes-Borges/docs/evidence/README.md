# Evidências

Esta pasta centraliza evidências de funcionamento, comparação de soluções e plano de commits.

- `funcionamento.md`: evidência operacional do fluxo (incluindo Supabase)
- `test-results.md`: status dos testes automatizados
- `comparacao-solucoes.md`: análise comparativa das 3 abordagens
- `plano-de-commits.md`: sugestão de commits por etapa avaliada
- `benchmark-output.json`: saída gerada por `src/run_benchmark.py`
