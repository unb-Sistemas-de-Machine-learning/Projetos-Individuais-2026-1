# Base de Conhecimento - Politicas de Triagem

## Categorias

### suporte_tecnico

Palavras-chave comuns:
- acesso
- login
- senha
- erro
- sistema
- bug

Encaminhamento sugerido:
- time de suporte tecnico.

### financeiro

Palavras-chave comuns:
- boleto
- pagamento
- taxa
- reembolso
- cobranca

Encaminhamento sugerido:
- time financeiro.

### comercial

Palavras-chave comuns:
- parceria
- proposta
- contrato
- servico

Encaminhamento sugerido:
- time comercial.

### outro

Usar quando nao houver indicios fortes das categorias acima.

## Regras de urgencia

- **alta:** interrupcao total do servico, prazo menor que 2h, termos como "urgente", "critico", "parado".
- **media:** impacto moderado e prazo de resposta no mesmo dia.
- **baixa:** demanda informativa ou sem bloqueio operacional.

## Casos sensiveis

Encaminhar para revisao humana quando houver:
- dados pessoais sensiveis;
- risco juridico/disciplinar;
- ambiguidade entre categorias.
