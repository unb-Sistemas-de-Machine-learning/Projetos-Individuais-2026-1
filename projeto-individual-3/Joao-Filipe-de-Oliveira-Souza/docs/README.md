# Documentacao do Projeto

Este diretorio concentra os artefatos de engenharia exigidos no projeto.

## Arquivos

- `mission-brief.md`: contrato de missao do agente.
- `mentorship-pack.md`: criterios de julgamento e padroes de qualidade.
- `workflow-runbook.md`: processo adotado para desenhar e validar a solucao.
- `merge-readiness-pack.md`: consolidacao de prontidao para revisao/merge.
- `adr/001-escolha-da-solucao.md`: decisao arquitetural da solucao final.
- `evidence/README.md`: orientacao para anexar evidencias de execucao.
- `evidence/evidence-log.md`: trilha de execucao dos testes e comprovacoes.
